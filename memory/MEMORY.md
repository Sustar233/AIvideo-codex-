## 用户偏好摘要
- 当前项目优先做人设提示词和分镜脚本；剧本分析、数据库维护、场景/道具梳理只做辅助。
- 角色等级不通过勋章/肩章视觉化
- 角色服装以现代日常装为主，不做废土朋克/做旧
- 人设图提供 2-3 版：精简版 + 完整版 + 可选风格版
- 人设图默认不加 negative prompt；特殊情况用中文“避坑说明”
- 人设提示词.md 只保留最新提示词，不累积旧版本
- 新增凶兽/灵体/重要道具时，必须同步生成视觉提示词并更新人设提示词.md
- 发现剧本严重问题时暂停，先问用户意见再继续
- 分镜默认支持迭代：可先出单元预览，再按反馈局部修改
- 多人场景启用独立优化流程：人数锁定、站位图谱、轴线锁定、景别降载
- 战斗分镜启用独立优化流程：动作切片、力学传导、能力源点锁定、三维轨迹和结果确认
- 用户说“简单”“简要”“简单解析”时，表示该集非其负责；仅做剧情归纳和必要问题提示，不生成人设、场景或其他视觉提示词
