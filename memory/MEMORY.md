## 用户偏好摘要
- 角色等级不通过勋章/肩章视觉化
- 角色服装以现代日常装为主，不做废土朋克/做旧
- 人设图提供 2-3 版：精简版 + 完整版 + 可选风格版
- 人设图默认不加 negative prompt，特殊情况用中文
- 人设提示词全部改用中文写入
- 人设提示词.md 只保留最新提示词，不累积旧版本
- 新增凶兽/灵体/重要道具时，必须同步生成视觉提示词并更新人设提示词.md
- 发现剧本严重问题时暂停，先问用户意见再继续
- 每集分析时生成角色设定简报（防OOC专用）
