# AIvideo | 《琥珀之城》——末日异能 AI 短剧项目

项目状态：第1-13集剧情概要已入库，数据库完整。

## 项目结构
- 数据库：剧本文件夹/database/（7个JSON文件）
- 剧本文件：剧本文件夹/（第1-13集）
- 人设提示词：剧本文件夹/人设提示词.md（只保留最新提示词，不累积旧版本）
- 自定义技能：.agents/skills/
- 项目记忆：memory/

## 世界观概要
- 城市：琥珀之城（旧称"白鸽之城"），鹰国首都，被琥珀屏障保护
- 屏障：潘诺西亚外星人的基因时停屏障，黄色蜂窝块碗状形态
- 制度：末日法典/夕阳法典，人分九等
- 凶骸：高级凶兽的骨质能源，通过手术赋予人类能力
- 融骸者：人类最高生态位（凶骸与人体融合）
- 黄宫：核心治理组织，7名仲裁官
- 浮梦组织：被琥珀军赶出城的起义军团

## 主要角色
- 于夜：9级背包客→融骸者→5级仲裁官"地震"，重力异能（魔律克之手）
- 蓝柚：粉发少女，浮梦战士，凶骸手术者
- 蔡浩（寸头男）：队长，小人得志
- 罗岚（鸣蛇）：蛇臂仲裁官
- 晏江：4级琥珀官，白发军人
- 赵可（风铃猫）：红发萝莉执行官
- 完整40个角色见 database/角色.json

## 项目规则（用户偏好）
1. 角色等级不通过勋章/肩章视觉化
2. 角色服装以现代日常装为主，不做废土朋克/做旧
3. 人设图提供2-3版：精简版 + 完整版 + 可选风格版
4. 人设图不加 negative prompt，特殊情况用中文
5. 人设提示词.md 只保留最新提示词，不累积旧版本（旧版用户自行保存到资产库）
6. 新增凶兽/灵体/重要道具时，必须同步生成视觉提示词
7. 发现剧本严重问题时暂停，先问用户意见
8. 分镜画面风格统一固定为：3D写实动漫风格，电影级三维动画质感，次世代CG渲染，高精角色模型，富有层次感的体积光影，电影级景深虚化，无明显2D勾线，虚幻引擎5渲染风格；除非用户明确指定例外

## 剧本处理工作流
1. 读取剧本 → 2. ScriptAnalyzer提取新元素，对比数据库
3. 更新JSON → 4. 生成提示词 → 5. 写入/覆盖到人设提示词.md
6. 如需分镜 → 使用storyboard-director

## 已安装技能
- ScriptAnalyzer：剧本分析，提取人设/场景/道具
- ai-short-drama-character：角色人设图片提示词生成
- storyboard-director：分镜头脚本生成
- qiuzhi-skill-creator：创建新技能
