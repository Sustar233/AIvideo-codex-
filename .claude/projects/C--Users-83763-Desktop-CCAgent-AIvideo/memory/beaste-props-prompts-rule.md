description: 新增凶兽和重要道具时，必须同步生成视觉提示词
metadata:
  type: feedback
---

用户要求在后续剧本处理中，对于新增的**凶兽/灵体**和**重要道具**，必须同步生成视觉设计图提示词，归类到 `人设提示词.md` 的凶兽/道具分类下。不能只记录数据库条目而不出图。
- 凶兽/灵体 → 放在 `人设提示词.md` 的「凶兽/灵体提示词」分类，输出版本A（精简必要版）+ 版本B（完整创意版）
- 重要道具 → 放在 `人设提示词.md` 的「道具提示词」分类，输出标准版

**Why:** 新出现的视觉元素需要配套生图提示词，方便后续制作参考。

**How to apply:** 每次处理剧本的"第三步·生成提示词"时，将新增凶兽和重要道具纳入生图范围，与人物/场景同等对待。
