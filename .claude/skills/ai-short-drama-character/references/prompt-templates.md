# 提示词模板库

## 通用结构模板

```
[A], [B], [C], [D], [E], [F]

A: 主体（性别 + 年龄 + 身材 + 姿态）
B: 面部（五官 + 发型 + 妆容 + 表情）
C: 服装（上装 + 下装 + 鞋子 + 配饰）
D: 场景（环境 + 背景元素）
E: 光影（光源 + 色调 + 氛围）
F: 技术（质量 + 风格 + 渲染参数）
```
