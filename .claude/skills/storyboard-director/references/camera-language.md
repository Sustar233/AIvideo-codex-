# 镜头语言与构图规则

## 一、镜头语言调用规则

镜头语言用于描述摄影机如何拍摄。它只负责"怎么拍"，不负责"人物怎么演"。

人物表情、微表情、本能反应、呼吸、手部动作和表情变化轨迹，必须写入"画面描述"栏，不要混进"镜头语言"栏。

**推荐格式：**

```
景别 / 机位角度 / 运镜方式 / 构图与焦点 / 镜头功能
```

示例：

```
镜头语言：中近景 / 平视略低机位 / 缓慢前推 / 焦点锁定人物眼睛，背景轻微虚化 / 用于突出听者压住情绪的反应
```

---

### 1. 景别选择规则

根据剧情功能选择景别，不要全用中景。

| 景别 | 功能 |
|------|------|
| 远景 | 交代环境和人物处境 |
| 全景 | 交代人物站位和空间关系 |
| 中景 | 表现人物动作和对话关系 |
| 中近景 | 兼顾表情和身体动作，适合短剧对话 |
| 近景 | 突出眼神、嘴角、呼吸、情绪变化 |
| 特写 | 强化关键表情、反应、台词落点 |
| 极特写 | 拍手指、喉结、眼睫、物件边缘、关键道具细节 |

**竖屏短剧优先使用：** 中近景、近景、特写、极特写。

远景和全景只用于交代环境，不要长时间使用，避免人物情绪不可读。

---

### 2. 机位角度规则

机位角度必须服务人物关系和戏剧压力。

| 角度 | 功能 |
|------|------|
| 平视 | 自然真实，适合正常对话 |
| 略低机位 | 增强人物气场和压迫感 |
| 低机位仰拍 | 用于主角高光、权力感、强压迫 |
| 略高机位 | 表现人物被压制、失落、无助 |
| 俯拍 | 表现处境弱势、空间压迫或被围观 |
| 侧拍 | 表现对峙、距离感、人物关系 |
| 越肩反打 | 表现对话关系和压迫方向 |
| 贴近前景遮挡 | 增强偷窥感、压迫感或空间层次 |

**注意：** 一个镜头只选择一个主要角度。不要同时堆叠"低机位 + 俯拍 + 环绕 + 旋转"。

---

### 3. 运镜方式规则

运镜必须服务剧情，不要为了炫技而动。

| 运镜 | 适用场景 |
|------|---------|
| 固定机位 | 压迫、沉默、冷场、情绪停顿 |
| 缓慢前推 | 情绪逼近、压迫增强、真相靠近 |
| 缓慢后拉 | 失落、抽离、关系疏远 |
| 侧向跟拍 | 人物移动、进入场景、关系推进 |
| 轻微手持 | 紧张、真实、慌乱、不稳定情绪 |
| 稳定横移 | 展示空间关系或多人物反应 |
| 低机位跟拍 | 主角登场、气场建立 |
| 遮挡转场 | 人物进门、转身、擦肩而过、情绪切换 |
| 快速甩镜 | 突然反应、意外发现、冲突爆点 |
| 微距静推 | 手指、物件边缘、关键道具、细节反应 |
| 焦点转移 | 信息揭示、物件到表情、说话者到听者 |
| 前景揭示 | 悬念、进入空间、角色登场 |
| 轻微环绕 | 关系变化、压迫空间、人物被围困感 |

**注意：** 每个镜头只保留一个核心运镜。不要写"推近 + 环绕 + 旋转 + 变焦 + 手持"这种混合动作。

---

### 4. 镜头运动丰富性规则

镜头运动必须服务剧情、情绪和信息揭示。

**核心原则：**
- 单个镜头只保留一个核心运镜。
- 单个单元内部要有运动节奏变化。
- 运镜必须对应明确的戏剧功能。

#### A. 对峙压迫型节奏

适合争吵、审问、质问、威胁。

```
固定建立关系 → 缓慢前推增强压力 → 反应镜固定停顿 → 极特写静态落点
```

#### B. 情绪崩塌型节奏

适合委屈、崩溃、失望、被揭穿。

```
稳定镜头 → 轻微手持 → 缓慢后拉 → 特写静止
```

#### C. 真相揭示型节奏

适合发现证据、打开文件、看到关键物件、听到反转。

```
遮挡或前景引入 → 焦点转移 → 快速反应切镜 → 静态特写落点
```

#### D. 主角高光型节奏

适合主角登场、反杀、身份揭示、气场建立。

```
局部跟拍 → 低机位前推 → 遮挡揭面 → 固定特写落眼神
```

#### E. 群像反应型节奏

适合众人围观、评判、议论、多人冲突。

```
全景建立关系 → 稳定横移扫过群体 → 单人反应固定镜 → 主体特写落点
```

---

### 5. 构图与焦点规则

Seedance 2.0 更容易理解"焦点锁定哪里"，因此镜头语言里必须尽量写清楚焦点和主体位置。

可用表达：
- 焦点锁定人物眼睛 / 焦点锁定手部动作
- 焦点从关键物件转到人物表情
- 前景轻微遮挡，主体清晰
- 背景轻微虚化但保留真实空间
- 主体居中压迫构图
- 人物站在画面三分线，保留眼神方向留白
- 双人对峙构图，中间保留冲突空间
- 竖屏构图中人物面部位于上三分线附近
- 头顶保留适度安全空间，避免五官贴边

**竖屏短剧特别要求：**
- 人物面部必须清晰可读。
- 情绪镜头优先保证眼神和嘴部可见。
- 不要让人物被背景吞没。
- 不要为了电影感把脸压得太暗。
- 不要过度虚化导致场景关系丢失。

---

### 6. 镜头功能规则

每个镜头语言最后可补一句镜头功能，帮助 AI 理解这个镜头为什么存在。

可用功能：
- 用于交代空间 / 用于建立人物关系
- 用于承接上一镜动作 / 用于突出听者反应
- 用于表现本能微反应 / 用于压迫感增强
- 用于情绪停顿 / 用于台词落点
- 用于揭示关键物件 / 用于主角高光登场
- 用于冲突爆点 / 用于转场衔接

镜头功能必须简洁，不写剧情解释，不写人物心理独白。

---

### 7. 焦段和光圈使用规则

焦段和光圈可以保留，但只作为辅助参考，不作为每个镜头的必填项。

| 焦段 | 推荐用途 |
|------|---------|
| 24mm | 交代空间和环境压迫 |
| 35mm | 人物与环境关系 |
| 50mm | 自然对话、正反打 |
| 85mm | 情绪近景、人物特写 |
| 100mm 微距 | 手部、物件、眼神细节 |

竖屏短剧中，不建议长时间使用 135mm 以上焦段。焦段信息如果影响提示词清晰度，可以省略。

**优先级：** 景别、机位、运镜、焦点、镜头功能 ＞ 焦段、光圈参数

---

## 二、构图规则

### 1. 9:16 竖屏构图

- 避免长期使用 135mm 以上长焦。
- 常规焦段控制在 24mm-85mm。
- 人物头顶保留约 15% 安全留白。
- 不要让五官贴边。
- 对话时保持眼神方向留白。
- 对峙时双方可占左右三分线。
- 情绪压迫时可短暂使用顶边构图，但不超过 2 秒。
- 人物面部和眼神必须清晰可读。
- 不能为了电影感牺牲手机端观看清晰度。

### 2. 16:9 横屏构图

- 强调空间关系。
- 可用前景遮挡制造层次。
- 对峙戏保留中间冲突空间。
- 群像戏注意站位清楚，不叠人。
- 空间调度优先通过摄影机运动完成，不新增演员走位。
