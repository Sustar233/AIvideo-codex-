# 《琥珀之城》第21集第6单元剪辑版

画面风格：
3D写实动漫风格，电影级三维动画质感，次世代CG渲染，高精角色模型，富有层次感的体积光影，电影级景深虚化，无明显2D勾线，虚幻引擎5渲染风格。精品16:9横屏电影，场景色调、环境物件、角色外观和能力特效必须依据原剧本、项目数据库与已有人设提示词生成，不得套用剧本外固定场景。画面保持清晰明暗层次和横屏构图；衣服和肢体保持符合角色设定的稳定形态，避免高穿帮变形。

[第21集-第6单元·剪辑版 / 猿山、于夜、赵可、罗岚、狼魁、凯旋]

第 6 单元：一秒睡着·候补镜头

场景：凶骸大赛会馆（一楼大厅及通向楼上的走廊）
时间：日/内
出场人物：猿山、于夜、赵可、罗岚、狼魁、凯旋

时长：15 秒

画面综述：
凶骸大赛会馆一楼走廊内，猿山位于方豪组一侧，与十米外站在晏组队伍前方的于夜短暂对视后转身朝楼上走去。赵可站在于夜身侧，抬手轻拍他的胳膊，提起于夜和猿山上次未完成的交手；于夜平静回应“碰上让他一秒睡着”。罗岚站在二人同侧后方，先捂嘴扑哧笑出声，再看向于夜评价他变幽默了。本候补版保持完整动作与台词顺序，改用楼梯高位全景、双人侧面中景、侧向双人跟拍、于夜正面中近景、微距上摇与反向越肩镜头提供逐镜替换选择。

镜号 1

镜头语言：全景 / 楼上通道入口斜俯机位 / 固定机位 / 猿山位于右侧近景并转向楼上，于夜位于十米外左侧背景，走廊纵深完整保留 / 独立建立十米对视结束后的离场动作

画面描述：【猿山始终位于走廊右侧，于夜保持在十米外左侧背景；猿山转身后只朝楼上通道移动，双方不再靠近】猿山的下颌仍略微绷紧，他把视线从于夜所在方向移开，肩部与重心依次转向楼上入口；随后迈步朝高位摄影机下方的通道走来，脏辫随转身短促摆动，于夜始终停在远处原位，不追赶、不释放能力。

台词：无。

音效：猿山转身脚步声 / 走廊环境回响 / 持续白噪音

镜号 2

镜头语言：双人中景 / 于夜与赵可侧面平视机位 / 固定机位 / 于夜位于左侧并望向猿山离开的方向，赵可位于右侧，焦点锁定赵可抬手轻拍于夜胳膊 / 以双人候补构图引出对话

画面描述：【以相机中轴线为基准，于夜始终站在左侧并面向楼上通道，赵可始终站在他右侧，二人不交换位置】赵可看着猿山离开的方向，眉尾轻轻抬起，随后抬手在于夜胳膊外侧连续轻拍两下；于夜的手臂自然下垂，身体没有因轻拍产生晃动，视线仍停在楼上通道方向。

台词：无。

音效：手掌轻拍声两下 / 远处脚步声 / 持续室内白噪音

镜号 3

镜头语言：双人中近景 / 走廊侧前方略低机位 / 缓慢侧向平移 / 赵可位于右侧主焦点，于夜侧脸位于左侧次焦点，罗岚、狼魁、凯旋保留在纵深后景 / 以移动双人候补镜承载赵可完整台词

画面描述：【以相机中轴线为基准，于夜始终在左、赵可始终在右，罗岚、狼魁、凯旋保持在二人同侧后景，所有人不改变站位】赵可收回轻拍的手，抬眼看向于夜，眉毛扬起后停住，一侧嘴角露出明显弧度；摄影机沿二人侧前方缓慢平移，她吸气很短，带着看热闹的兴致开口，于夜保持安静聆听。

台词：赵可【带着看热闹的兴致，正常语速】：“上次没打完，待会儿看你俩能不能碰上。”

音效：赵可清晰说话声 / 猿山脚步声渐远 / 持续室内白噪音

镜号 4

镜头语言：中近景 / 于夜正面平视机位 / 固定机位 / 于夜占据画面中心，赵可侧脸轮廓保留在右侧边缘，焦点锁定于夜从楼上通道移向赵可的视线与嘴部 / 用正面候补镜落下于夜回应

画面描述：【于夜始终站在画面中心并保持身体朝向楼上通道，赵可位于他右侧边缘；于夜只转动视线和头部，不改变站位】于夜听完后眨眼一次，眼神从猿山离开的方向移到赵可脸上，头部随视线轻偏；他的呼吸没有变化，嘴角短暂抬起，停顿半秒后以平稳语气开口。

台词：于夜【平静笃定，正常语速】：“碰上让他一秒睡着。”

音效：于夜平稳说话声 / 场馆远处底噪 / 持续室内白噪音

镜号 5

镜头语言：特写 / 罗岚正面手部起始机位 / 缓慢上摇 / 焦点先锁定罗岚抬起的手指，再随镜头上摇转至弯起的眼睛与被手遮住的嘴部 / 以微距候补视角捕捉扑哧笑声

画面描述：【罗岚始终站在于夜与赵可同侧后方并面向二人，不移动脚步】听到于夜的回答后，罗岚的手指先在身侧轻轻收拢，随后抬到嘴前；摄影机随动作上摇，她的眼睛逐渐弯起，肩线轻颤一下，短促呼吸从指间漏出，随即发出一声没有压住的扑哧笑声。

台词：无。

音效：罗岚短促扑哧笑声 / 抬手动作轻响 / 持续室内白噪音

镜号 6

镜头语言：近景 / 于夜右后方反向越肩机位 / 固定机位 / 于夜肩部虚化在右侧前景，焦点锁定罗岚放下手后看向于夜的眼睛与嘴部 / 以反向越肩候补镜结束单元
