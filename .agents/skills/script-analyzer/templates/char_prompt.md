# 角色提示词模板（中文版）

每次为角色生成生图提示词时，按以下三段式输出：

## 必含要素（中文描述格式）

| 字段 | 说明 | 示例 |
|------|------|------|
| **姓名** | 角色中文名 | 于夜 |
| **种族/身份** | 人类、外星种族、组织成员等 | 人类，琥珀之城守卫者 |
| **外貌特征** | 脸型、眼睛、发型、发色、肤色、身高体型 | 棱角分明的下颌线、黑色刺猬头、琥珀色瞳孔、高挑瘦削 |
| **服装风格** | 主色调、材质、标志性配饰 | 带琥珀滚边的黑斗篷、发光腕甲 |
| **表情气质** | 整体氛围感 | 沉着冷静、算计的眼神 |
| **构图模式** | 主要角色默认 `[三视图]`，路人角色默认 `[普通全身图]` | 三视图转画 / 全身肖像 |

## 输出格式

每次为角色生成提示词时，按以下三段式输出：

```
【角色名】- [构图模式]

版本A（精简必要版）：
> prompt: [只包含剧本明确写出的特征——发色、体型、服装款式等，不添加任何推测性描述]

版本B（完整创意版）：
> prompt: [在必要元素基础上补充合理推断——气质神态、光影氛围、色彩层次、材质细节等]

负面提示词：
[从 style_guide.md 复制负面列表]
```

### 注意事项

- **所有提示词必须使用纯中文**，不要夹杂英文
- **版本A原则**：只保留剧本/数据库中**明确出现**的角色特征，绝不自行脑补。例如剧本写了"粉发少女"就写"粉色长发、娇小体型"，但不该写"星空瞳孔""神秘气场"这类未提及的元素
- **版本B原则**：以版本A为基础展开。允许根据角色身份（如龙骧/背包客/裁决官）、所属世界观风格（赛博朋克+废土）、已知能力（治愈/异型臂）做合理的视觉推演
- 同一角色在不同集数若换装，标注"第X集更换为..."
- 新增角色需在输出末尾附上 JSON 片段，格式与 `角色.json` 一致，方便直接追加
- **构图模式自动分配**：主要角色（有名字、有台词、推动剧情）默认出三视图；路人角色（无名字、仅背景出现）默认出全身图
