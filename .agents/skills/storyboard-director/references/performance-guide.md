# 表演细节参考手册

## Seedance 2.0 可读版

本文件用于指导演员表情、身体状态、情绪变化和镜头内可见表演的设计。

---

## 一、画面描述公式

每个镜号的画面描述必须使用以下结构：

**角色主体 + 情绪氛围锚点 + 眼神方向 + 面部变化 + 呼吸/停顿 + 手部或身体动作 + 表情变化轨迹 + 客体对象/方位**

示例：

> 角色压着怒意站在原地，视线先落在桌面上，没有立刻看人，眼皮微微压低，嘴角一点点收平，吸气时下颌线绷住，右手指节扣紧桌沿，最后缓慢抬眼看向对方，表情从克制变成冷硬。

**禁止写：**

> 他眼神里写着"我不想解释"。

**应该写：**

> 他没有看对方，眼皮微垂，视线斜压在地面，嘴角没有任何笑意，吸气时脸颊微微收紧，吐气后停了半秒，最后仍然没有开口。

---

## 二、表演描述基本公式

重要情绪镜头建议使用以下公式：

**情绪氛围锚点 + 眼神方向 + 面部变化 + 呼吸/停顿 + 手部或身体动作 + 表情变化轨迹**

---

## 三、情绪氛围词使用规则

可以使用以下情绪词作为表演方向锚点，但每个情绪词后面必须立刻跟随具体可见的面部、眼神、呼吸、手部或身体动作：

压抑、冷硬、克制、委屈、心虚、慌乱、错愕、羞恼、轻蔑、疲惫、失望、崩溃边缘、上位者压迫感、被羞辱后的沉默、强撑镇定、暗中观察、不动声色、情绪被击穿、怒意压在脸上、局促不安、冷场后的尴尬、期待落空后的僵住、爆发后的余震、被误解后的沉默、强忍不发作、体面快要挂不住、突然意识到不对劲。

**错误写法：**

> 他很愤怒，气场全开。

**正确写法：**

> 他压着怒意站在原地，眼神没有马上抬起，嘴角慢慢收平，下颌线绷紧，手指扣住身边的物件，吸气停了一拍后才看向对方。

## 四、Seedance 2.0 可读的表演结构

### A. 眼神方向

**优先写清楚眼睛看哪里。**

可用表达：
- 视线落在桌面上 / 视线压在地面
- 眼神从关键物件移到对方脸上
- 先看向门口，再看回对方
- 眼睛没有直视对方 / 视线短暂闪开
- 眼神停在对方手上 / 抬眼看向对方
- 视线微微下垂 / 眼神越过对方看向远处
- 视线在众人脸上扫过一圈
- 眼睛停住，短时间没有眨动

**避免写：**
- 眼神里写着不甘 / 眼神充满故事
- 眼神说明了一切 / 眼神像在说某句话

### B. 眉眼状态

可用表达：
- 眉心慢慢收紧 / 眉尾轻轻压低
- 眉毛先抬起，随后僵住
- 眼皮微微压低 / 眼眶发紧，但没有落泪
- 眨眼频率变慢 / 眼睛短暂睁大
- 眼神失焦半秒 / 眼睫轻轻颤动
- 眼角轻微抽动 / 眉头松开后又重新压住

**避免只写"眼眶微红"**，如果必须写，需要补充动作：

> 她眼眶发紧，眨眼变慢，视线短暂垂下，嘴唇抿住后又松开，像是把话压了回去。

### C. 嘴部与下颌

可用表达：
- 嘴角一点点收平 / 嘴唇抿成一条线
- 嘴唇微张却没有出声 / 嘴角轻微抽动
- 一侧嘴角短暂抬起 / 下颌线绷住
- 牙关轻轻咬紧 / 喉结轻轻滚动
- 说话前嘴唇动了一下又停住
- 呼气时嘴角微微下压
- 笑意只停在嘴角，眼睛没有跟着笑

**避免直接写：** 冷笑、邪笑、崩溃大哭。除非剧本明确需要，否则要改成可控的表演细节。

### D. 呼吸与停顿

可用表达：
- 吸气变短 / 呼吸停了一拍 / 吐气很轻
- 说话前短暂停顿 / 声音出口前先压住一口气
- 胸口轻微起伏 / 鼻翼轻轻动了一下
- 呼吸从平稳变得急促 / 话到嘴边停住
- 尾音压低 / 停顿半秒后才开口

### E. 手部微动作

可用表达：
- 指尖扣住桌沿 / 手指轻轻收紧
- 手背青筋短暂浮起 / 指节压白
- 拇指摩挲物件边缘 / 手指停在半空
- 握住物件的手慢慢收紧 / 手里的物件停住
- 掌心压在桌面上 / 手指整理袖口
- 攥住衣角后又松开 / 手掌从桌边缓慢收回
- 指尖轻敲桌面一次后停住

**避免无意义堆叠：** "眼神压低、下颌绷住、手指收紧、呼吸停住" 这类组合不能反复套用，必须根据角色和剧情重新设计。

### F. 身体姿态

可用表达：
- 身体站得很稳 / 肩膀轻轻塌下去
- 身体微微后撤半步 / 上半身保持距离感
- 下巴轻轻抬起 / 头部略微偏向一侧
- 背脊挺直，但手指已经收紧
- 身体僵住，没有继续动作
- 原本要转身的动作停住 / 重心压在后脚
- 身体向桌面靠近一点 / 肩线从紧绷慢慢松掉

## 五、反应镜头的微表情与本能肢体反应规则

反应镜头不是简单写"震惊""愤怒""难过"，而是要优先捕捉人物在受到外部刺激后，先于语言和完整表情出现的本能反应。

### 反应镜头总原则

描写顺序：**刺激进入 → 本能反应 → 意识控制 → 最终表情落幅**

1. 先写人物被什么刺激打到。
2. 再写身体第一时间的自然反应。
3. 再写他试图控制、掩饰、压住或转化。
4. 最后落到清晰表情和姿态。

**错误写法：** 她很震惊。/ 他十分愤怒。/ 她很难过。

**正确写法：**

> 她听到最后一句时，手上的动作先停住，眼睛短暂睁大，呼吸卡了一下，嘴唇张开却没发出声音，随后视线快速移向说话的人，眉心慢慢收紧，错愕转成压着火的难堪。

### 本能反应优先级

在反应镜头里，优先写以下 5 类本能反应，不要一上来就写完整情绪结论：

#### A. 动作中断反应

- 正要转身的动作停住 / 刚要开口却停住
- 原本抬起的手没有落下 / 身体刚要坐下又顿住
- 整理衣服的动作突然停住
- 正在写字的笔尖停在纸上
- 原本要走的脚步停了一下 / 手里的物件停在半空

#### B. 呼吸与停顿反应

- 呼吸停了一拍 / 吸气变短 / 吐气变轻
- 胸口起伏突然变明显 / 鼻翼轻轻动了一下
- 说话前先压住一口气 / 话到嘴边停住
- 尾音突然发虚 / 呼吸由平稳变得急促
- 喉结轻轻滚动后才开口

#### C. 视线本能反应

- 视线先停住 / 眼神短暂失焦
- 视线先滑向地面 / 先看向门口，再回到对方脸上
- 眼睛快速扫过周围人的反应
- 眼神闪开，不敢立刻对视 / 抬眼只看了一下又躲开
- 目光停在对方手上 / 眼睛没有眨，像是一下没反应过来

#### D. 嘴部与下颌失控反应

- 嘴唇张开却没出声 / 嘴角轻微抽动
- 嘴角原本的笑意挂不住 / 嘴唇抿住后又松开
- 下颌线突然绷紧 / 牙关轻轻咬住
- 说话前嘴唇动了一下又停住 / 一侧嘴角僵住
- 原本的笑只停在嘴角，眼睛没有跟上
- 嘴角往下压了一点又被强行控制住

#### E. 手部与支撑反应

- 手指扣住桌沿 / 指尖突然收紧
- 手里的物件握得更紧 / 手背青筋浮起一瞬
- 手掌压在桌面上 / 指尖摩挲物件边缘
- 攥住衣角 / 手指停在半空
- 突然扶住椅背 / 轻轻按住膝盖
- 手掌从桌边慢慢收回来

## 六、按景别选择表演细节

### 远景 / 全景

重点写：身体站位、空间距离、重心变化、肩背状态、人物与环境关系。
**不要写**太细的眼皮、瞳孔、嘴角。

### 中景

重点写：头部角度、视线方向、手部动作、身体姿态、基础表情变化。

### 近景 / 特写

重点写：眼神、眉眼、嘴唇、下颌、呼吸、表情变化轨迹。

### 极特写 / 微距

重点写：指尖、喉结、眼睫、物件边缘、衣料、手部细节、关键道具细微变化。

---

## 七、表情变化必须有过程

重要情绪镜头不能只写静态表情，必须写出变化轨迹。

**推荐结构：** 起始状态 → 刺激反应 → 情绪压制或爆发 → 落幅状态

可用变化轨迹：
- 从冷静到压怒 / 从错愕到羞恼
- 从轻蔑到警觉 / 从强撑到崩溃
- 从期待到失望 / 从心虚到强装镇定
- 从委屈到放弃解释 / 从惊讶到立刻掩饰
- 从疲惫到冷漠 / 从沉默到爆发前的压制

---

## 八、禁止机械套用

以下词可以使用，但不能在多个镜头里反复固定组合：

眼皮压低、下颌绷住、指尖收紧、喉结滚动、嘴角抽动、眼眶发紧、呼吸停一拍、肩膀塌下去。

每次设计表演时，必须结合以下 7 个维度重新设计，不得直接复制参考句：

1. 当前角色身份
2. 当前人物关系
3. 当前台词内容
4. 当前镜头景别
5. 当前情绪阶段
6. 当前动作任务
7. 当前场景氛围

---

## 九、Seedance 2.0 禁止写法

禁止写：
- 他心里想着…… / 她眼神里写着……
- 他仿佛在说…… / 她像是灵魂被抽空……
- 他的气场压过全场…… / 杀气弥漫全场……
- 世界安静了…… / 时间仿佛停止……
- 他知道自己很强但懒得解释……
- 她整个人破碎了…… / 他像被雷劈了一样……
- 空气突然凝固…… / 情绪在空气里爆开……
- 眼神说明了一切…… / 内心仿佛在怒吼……

这些表达容易让模型误判成抽象、旁白、字幕、特效或不可见情绪。

---

## 十、Seedance 2.0 推荐写法

优先写：
- 眼睛看向哪里 / 视线如何移动
- 眉眼如何变化 / 嘴角如何变化
- 下颌是否绷住 / 呼吸是否停顿
- 手指是否收紧 / 身体重心是否改变
- 动作是否中断
- 表情从什么状态变到什么状态
- 最后落在什么表情或姿态上
- 人物受到刺激后的第一本能反应是什么
- 人物如何试图控制、掩饰或压住这个反应

**核心原则：**
- 所有潜台词都必须转化成可见表演。
- 所有情绪氛围都必须落到身体和面部细节。
- 所有表情变化都必须服务当前剧情，不得为了堆细节而堆细节。
